PROJETOS

Pokedex

O projeto Pokedex do curso Labenu é um exercício prático de desenvolvimento web que tem como objetivo ensinar os alunos a utilizar tecnologias front-end, como React.js, JavaScript, HTML, e CSS, para construir uma aplicação interativa e visualmente atraente.
O principal objetivo do projeto é criar uma aplicação de Pokedex, semelhante à da famosa série Pokémon, que permite ao usuário visualizar uma lista de pokémons, explorar detalhes de cada um, como habilidades, tipos, e status, além de adicionar ou remover pokémons de uma lista de favoritos.

Viva Música

O Viva Música é uma landing page para divulgar aulas de música para crianças e adultos. Criada com HTML, CSS, e JavaScript, a página oferece uma experiência visual atraente e informações claras sobre os cursos disponíveis.
O objetivo do Viva Música é praticar habilidades em HTML, CSS, e JavaScript, explorando conceitos de design responsivo e interatividade, além de aplicar boas práticas de desenvolvimento web.
